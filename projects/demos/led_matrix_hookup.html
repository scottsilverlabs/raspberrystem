LED Matrix cell

Need:

  • 1 LED matrix module
  • 1 LED matrix cable
  • 7 jumper wires

Setting up the cell:

  1. Place the LED matrix cell on the breadboard.
  2. Using the LED matrix cable, connect the 6 pins from the LED matrix to the breadboard.
  3. Using 6 of the jumper wires, connect each wire from the breadboard to the Raaspberry Pi.
    [insert wiring diagram here!]
  4. Using the last jumper wire, connect the 3 left most socket of the last matrix to the 9/MISO port on the Raspberry Pi.
    [insert wiring diagram here!]

Running the program:

  1. Click the folder icon on the top left of the screen.
  2. Select the program "walking_sprite_easy.py"
  3. Click the play button on the top left of the screen.
  4. You will now see an animated guy walking on the LED matrix
